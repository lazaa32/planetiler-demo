# [Flatmap Demo](https://onthegomap.github.io/flatmap-demo)

Demo website showing vector tiles built by the [Flatmap](https://github.com/onthegomap/flatmap) project using the [OpenMapTiles](https://github.com/openmaptiles/openmaptiles) schema and [MapLibre GL JS](https://github.com/maplibre/maplibre-gl-js).

See it [here](https://onthegomap.github.io/flatmap-demo)
